08. Recurrent Neural Networks

Posted on 2020-05-25 Edited on 2020-08-07 In 计算机科学与技术 , 计算机视觉 (Stanford CS231n)

Common models

  1. One to one model: vanilla model
  2. One to many: image captioning
  3. Many to one: sentiment classification
  4. Many to many (some input without output while some output without input): machine translation
  5. Many to many (each input has an output): vedio classification on frame level

Recurrent Neural Network

  1. It has a recurrent core that will update each time it’s needed, and the next data will still be feed to it.
  2. $h_t = f_W(h_{t-1},x_t)$
