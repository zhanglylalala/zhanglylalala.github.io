汉诺塔

递推求步数和方案
有一根1号柱子上有从上到下半径递增的圆盘，要把这些圆盘(可以再借助一根2号柱子)挪到另一根3号柱子上，且挪动过程(包括结果)中，小圆盘始终在大圆盘上面。
已知，挪动的方法就是先把最上面的n-1个圆盘放到2号柱子上，再把底下最大的那个圆盘放到3号柱子上，这时问题就变为把圆盘从2号柱子借助1号柱子，放到3号柱子上


递推求步数
注意到把n - 1个圆盘从一个柱子挪到另一个柱子的步数和从哪个柱子挪到哪个柱子无关，是固定的所以可以用递推

数学求步数
从递推公式可以得到：a~n~ = 2 * a~n-1~ + 1，a~0~ = 0，所以a~n~ = 2^n^ - 1

计算组合数

组合数C(m, n) = m! / (n! * (n - m)!)，原始公式涉及到三个阶乘，不但计算复杂，还容易爆long long，利用公式C(m, n) = C(m, n - 1) + C(m - 1, n - 1)就可以利用记忆化+递归，或递推快速不爆上界地求出组合数

约瑟夫环

用数学代替模拟，时间复杂度由暴力模拟的O(n * m)变为了O(n)，但不能求出出列的顺序，只能得到最后出列的人

概率求解

1、对于独立重复实验中，重复n次实验，某一概率为p的事件恰好出现m次的概率：
P = C(m, n) p^m^ p^n-m^
2、条件概率：在事件A发生的条件下事件B发生的概率P(B | A) = P(AB) / P(A)
