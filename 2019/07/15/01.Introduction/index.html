01. Introduction

Posted on 2019-07-15 Edited on 2020-08-07 In 计算机科学与技术 , Swift开发 (Stanford CS193p)

Creat a UIButton

Creat And Edit A Button

  1. Open the Main.storyboard file, and click the libary button to choose a UIView in it.
  2. We can edit the content in the UIButton by double click the button
  3. We can edit what the button looks like by choosing the button first and edit it at the Attributes Inspector when the top of it shows ‘button’ (if we click the background of the phone, it shows ‘view’)

Connect the Button with Code

  1. Choose the button
  2. Hold on the ctrl key and drag from the button to the viewController.swift wherever you want to put the code
  3. Then a window like below will show.
    The first label is connection, where we will choose Action.
    The second label is Object, which is view controller.
    The third one is name.
    The forth one is what kind of UIView is this, which is UIButton.
    The fifth one is at what circumstances will the system triggle the button, which is touch up inside.
    The sixth one is the argument, which can be none, sender or sender and event, at this case, we only want to know which the card is flipped over, so we choose sender(sender has every information about the button we just touched)
  4. After that, a method is created in the ViewController class, and now we have creat a UIButton and connect the button to the controller.

Flip the Card

We just need to define a function to change the status of the button every time we touch it to “flip” the card

Argument(s) We Need

To change the status, we need a target status and we need to know which button was touched. So we need two argument, one is the target status type of String, one is the touched button type of UIButton

Judge Which Face of the Button Is Over

Use the button.currenTitle to get the curren content of the button, if the content is empty, then we should change to the emoji face; if the content is the emoji, then we should change to the back of the card

Change the Appearence of the Button

  1. Set the string in the button by button.setTitle(_, for:). The first parametre is the string we want to set, the second parametre is UIControlState.normal.
  2. Set the background color by setting button.backgroundColor. We can assign a color literal to it by typing Color Literal and choose a color.

Call the Method

In the button method we defined in the last step, we call the flip-method. We pass the sender as the button parameter.

Show the flip count

We don’t need to touch the flip count, so a UILabel will work.

Creat A Label

  1. Drag a label out of the library and put it in the place we want
  2. Ctrl and drag the label to the code
  3. This time, we don’t need a function to do something when we touch the button, so we will set the connection to outlet, which will creat an instance property

Update the Counter

  1. First of all, we need to add one to the counter every time we touched a button
  2. We also need to update what shows in the screen, so we assign a string to the flipCountLabel.text

Update the flipCountLabel

In every button method, we will increment the flipCount, should we write the code after that? If we do so, the code will be very long, so we choose to use the property observer

Less button methods

  1. If we only use the way introduced at the beginning, and we also want every button send different messages, we will have to creat button methods as many as buttons. That’s annoy. So we are going to do something about that.
  2. The solution is to creat a button array.
  3. Drag a button to the view controller, and this time, we will creat neither a method, nor an instance property, but an array, so the connection we choose is the outlet collection, which will creat a button array.
  4. We connect all buttons to the button array and one button method. In that button method, we can use the index method to find the subscript of the touched button, and send the message in the message array with the correspond subscript
