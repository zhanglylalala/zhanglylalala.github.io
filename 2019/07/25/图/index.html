图

Posted on 2019-07-25 Edited on 2020-08-07 In 计算机科学与技术 , 数据结构 (清华大学邓俊辉)

术语

图用G = (V; E)表示，V表示顶点集，E表示边集

边

  1. 邻接关系：节点之间的关系
  2. 关联关系：节点与边之间的关系
  3. 若边的两个顶点没有先后关系，就把那条边称为无向边，所有边都是无向边的图称为无向图
  4. 若边的两个顶点有先后关系，就把那条边称为有向边，所有边都是无向边的图称为有向图
  5. 即有有向边，又有无向边的图称为混合图
  6. 无向边可以拆为两条方向不同的有向边
  7. n个顶点的无向完全图，有$\frac{n(n-1)}{2}$条边，n个顶点的有向完全图中有$n(n-1)$条边，n个顶点的强连通图至少$n$个顶点(形成一个大环)

路

  1. 简单路：每个顶点只出现一次(不一定要包含所有顶点)
  2. 环路也分为简单环路和环路
  3. 欧拉环路：每条边恰好出现一次(要包含所有边)
  4. 汉密尔顿环路：每个顶点恰好出现一次(要包含所有的点)

图的实现

  1. 邻接矩阵：若有n个节点，就建立一个n行n列的矩阵，如果节点i和节点j之间是邻接关系，就把第i行第j列的值设为1(无权图)或设为权值(带权图)
  2. 关联矩阵：若有n个节点，e条边，就建立一个n行e列的矩阵，如果节点i和边j之间是关联关系，就把第i行第j列的值设为1(无权图)或设为权值(带权图)
  3. 无向边的邻接矩阵要把每条边存储两遍，有向边只要存一遍

实现

  1. 一个一维向量存储所有的顶点，一个二维向量存储所有的顶点之间的连边情况
  2. 连边情况的存储用的是指针，指向一个存储了边的信息的结构体/对象，默认为NULL，表示没有连边
  3. 加边：新建一个边的信息，并用一个指针指向它，并将指针存储在边的二维矩阵中，注意点出入度的增加
  4. 删边：释放边的指针指向的内容，将边的指针设为NULL，减少点的出入度
  5. 加点：给一位顶点向量加一个元素，二维向量的每一位最后都加一个空元素为原先的点到新点的边的情况(一开始为空)，二维向量还要再加一行，表示新点到其余点的边的情况(一开始也为空，且数量和现点数一样多，比原点数多一个)
  6. 删点：删除顶点向量中对应的元素，二维向量中将其他点与它的连边删除，并移出二维向量，再把对应的那一维整个删除移出，修改所有点的出入度

标记边

  1. 树边：从当前点指向一个尚未访问的点的边
    [当结束点尚未访问]
  2. 后向边：从一个点指向其祖先的边(要回溯才能经过树边达到另一个点的边)
    [当结束点被访问过，但还在访问其后代]
  3. 前向边：从一个点指向其后代的边(要正向进行深搜才可以经过树边达到另一个点的边)
    [当结束点被访问过，且访问时间晚于当前点]
  4. 跨边：跨接两个无法经过树边达到的点的边
    [当结束点被访问过，且访问时间早于当前点]
  5. 后向边的出现就表示了图中有环路

DFS序

  1. 标记时间：第一次访问一个节点的时间为dTime，访问完当前节点的所有邻居时的时间fTime(就算没有邻居要访问时间也要加一)
  2. DFS序：按访问顺序排列，每个节点出现且只出现两次，一次对应dTime，一次对应fTime
  3. 括号引理：在DFS序中，被一个节点的两次出现夹在中间的所有节点都是这个节点的后代，否则就没有关系
    若u是v的祖先 ⟺ dTime(u) < dTime(v) && fTime(v) < fTime(u)
    若u是v的后代 ⟺ dTime(u) > dTime(v) && fTime(v) > fTime(u)
    否则u和v没有关系
