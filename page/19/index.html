React to touches

  1. We can react to touches with raw touch events. (touches, moves, up, etc.)
  2. But, we can also react to certain, predefined “gestures”. Gestures are recognized by instances of UIGestureRecognizer.
  3. There are two sides to using a gesture recognizer.
    First, add a gesture recognizer to a UIView
    Second, provide a method to “handle” that gesture
  4. Usually the first is done by a Controller. Though occasionally a UIView will do this itself if the gesture is integral to its existence.
  5. The second is provided either by the UIView or a Controller.

快速排序

过程

  1. 选取一个轴点，轴点一般就选为第一个元素
  2. 重新划分数组，使轴点左边的元素都不大于轴点；轴点右边的元素都不小于轴点
  3. 递归地继续排序轴点左边和右边的两部分

重新划分1

  1. 第一个元素被选取为轴点后就被备份起来，方便之后比较
  2. 备份了第一个元素之后，第一个位置就相当于空闲起来了
  3. $lo$指针指向待划分数组部分的开头，$hi$指向结尾，初始时$lo$不必为了轴点后退一格

串

Posted on 2019-08-02 Edited on 2020-08-08 In 计算机科学与技术 , 数据结构 (清华大学邓俊辉)

串

  1. 相等：$S[0, n) = T[0, m)：n = m$，且对应字符均相同
  2. 子串：$S.substr(i, k) = S[i, i + k)$，即从$S[i]$开始的连续$k$个字符
  3. 前缀：$S.prefix(k) = S.substr(0, k) = S[0, k)$，即最靠前的$k$个字符
  4. 后缀：$S.suffix(k) = S.substr(n - k, k) = S[n - k, k)$，即最靠后的$k$个字符
  5. $S.substr(i, k) = S.prefix(i + k).suffix(k)$
  6. 空串：$S[0, n = 0)$，是任何串的子串、前缀、后缀

优先级队列

Posted on 2019-08-01 Edited on 2020-08-07 In 计算机科学与技术 , 数据结构 (清华大学邓俊辉)

优先级队列

  1. 优先级队列是一种队列，是一种受限的数据结构，优先级队列中的元素都有一个优先级，每次出队的都是优先级最高的元素，能访问的也只有优先级最高的元素
  2. 队列和栈其实都是特殊的优先级队列。对于队列，越早入队，优先级越高；对于栈，越晚入栈，优先级越高
  3. 优先级队列要求能够插入值、查找优先级最大元素、删除优先级最大元素
  4. 如果用向量/列表实现优先级队列，插入只要O(1)，查找和删除都要O(n)
    如果用有序向量/有序列表，插入要O(n)，查找和删除要O(1)
    如果用BBST，实现过于复杂，功能过于强大

散列

二叉搜索树

Posted on 2019-07-26 Edited on 2020-08-08 In 计算机科学与技术 , 数据结构 (清华大学邓俊辉)

原理

二叉搜索树(BST)借鉴于列表和有序向量
BST的每个节点又关键码和数据组成，整个节点内的信息封装成一个词条类。这里要求关键码可以比较
在数据也可以比较的情况下，有时也直接把数据拿来做关键码
BST处处满足顺序性：一个节点的所有左后代的关键码都不大于该节点的关键码，一个节点的所有右后代的关键码都不小于该节点的关键码
BST的中序遍历序列就是由所有的节点组成的一个有序向量，中序遍历就是树的垂直投影的序列
