Posted on 2019-11-09 In ACM , Graph , Shortest Path

附加权值、输出路径、路径种数(题目保证唯一)

PTA L2-001
题目要求：
 1、输出最短路径的种数
 2、输出最多救援队人数
 3、输出最优路径
解决：
 1、最短路算法选择：Dijkstra算法
 2、路径种数：建立一个一维way数组，表示从出发点到一点的路径种数。初始化为0，出发点为1。在Dijkstra算法中，进入更新各个点到出发点距离时，若dist[i] < dist[x] + map[x][i]，则使way[i] = way[x]，若dist[i] == dist[x] + map[x][i]，则way[i] += way[x]
 3、输出最多救援队人数：建立一个一维数组men，表示从出发点到一点的最短路径上能有的最多救援队人数，再建一个一维数组people，存放各个点的救援队人数。关注人数意味着最优路径多了一个指标，故在Dij算法最开始找当前最优点时不仅仅要距离最短，也要人数最多，故需要增加一个指标来标记最多人数。更新各点距离时，若dist[i]==dist[x]+map[x][i]&&men[i] < men[x]+people[i]，则men[i]=men[x]+people[i]；若dist[i] < dist[x] + map[x][i]，则men[i]=men[x]+people[i]

 4、输出最优路径：从终点开始DFS，若当前点的总人数减掉当前点的单点人数等于第i个点的人数，且出发点到当前点的距离减掉当前点到第i点的距离等于出发点到第i点的距离，则第i点就是最短路中当前点的父节点，并以这点继续DFS，直到第i点为出发点

一、单源最短路——Dijkstra算法

单点出发的最短路问题，问一个点到另一个或多个点到最短路问题
思想：
 1、数据存储：一个二维数组map存储两点之间的距离， 一个一维数组dist存储两点之间的最短距离，一个一维数组vis标记一点是否已访问

 2、初始化：map和dist都先初始化为无穷大，如：0x7fffffff，vis初始化为0。若起始点为start，则先使dist[start] = 0。若点i和点j之间有连边，且距离为d，则map[i][j] = map[j][I] = max(map[i][j], d)。注意，不设vis[start] = 1

 3、处理：从所有点中，选出未被标记且离出发点最近的点，更新所有与这个点相连的点距离出发点的距离，使之最短。不断更新，直到所有点都被标记

二、含负权单源最短路——Bellman-Ford算法

这个算法的特点就是可以处理Dijkstra不能处理的负权
一个图如果有环，就只有三种环——零环、正环、负环。若一个图存在最短路则不含有任何环，因为若一个图里包含负环则不存在最短路；若最短路包含正环或零环则路径不会变长，甚至会变短
思想：
 数据存储：和Dijkstra一样，不用vis


 初始化：和Dijkstra一样，不用初始化vis

 处理：遍历所有边，计算出从一点，经过那条边到终点的最短路

如何判断负环？
最后再 遍历一边所有边，如果dist[i] + map[i][j] < dist[i]则说明有负环

三、Bellman-Ford算法的优化——SPFA算法

优化思路：在Bellman-Ford算法的基础上再增加一个队列来维护那些到出发点距离可能改变的点。
数据存储：在Bellman-Ford的基础上再加一个一维数组vis来标记点是否在队列内，一个一维数组enqueue_num来标记一点的入队次数


初始化：vis和enqueue_num初始化为0，其他同Bellman-Ford算法

处理：
 1、初始时候把出发点推入队列
 2、出队队首元素，判断所有以该点结尾的边，若到出发点的最短距离改变了且另一端点不在队列内，则把该点推入队内
 3、若一个点的入队次数大于点数，则直接退出算法且这时说明有负环

这个算法还有一个小优化：利用Bubble sort里的方法，在循环时加一个变量，循环到不再有值改变为止：如果变量值为真，就停止循环，变量初始化为假，每次循环的开头就把变量设为真，一旦修改一个地方的值就把变量改为假
假如只是单纯想判断是否有负环(如差分约束)，可以把所有点都先入队再跑

四、多源最短路——Floyd算法

多点出发的最短路算法，最好写的一个，近乎万能，也可以处理负权
思想：
 数据存储：一个二维数组d存储距离

 初始化：d[i][i] = 0，其他值为无穷大INF

 处理：a点到b点只有两种办法，直接从a到b，或a到某点c再到b，而a到b的距离就是这两种中的最小值。故我们利用三重循环，遍历所有可能的a，b，c，得到所有出发点到所有终点的最短路

若只关心i和j之间是否有通路？
把d[i][j] = min(d[i][j], d[i][k] + d[k][j]);改为d[i][j] = d[i][j] || (d[i][k] && d[k][j]);
注意，k是作为中转站，放在最外层循环，整个程序相当于查找是否存在以k为中转站，从i到j的最短路

最长路

最长路不能用Dijkstra，只能取负权之后用Bellman-Ford、SPFA、Floyd求最短路

已知，挪动的方法就是先把最上面的n-1个圆盘放到2号柱子上，再把底下最大的那个圆盘放到3号柱子上，这时问题就变为把圆盘从2号柱子借助1号柱子，放到3号柱子上


递推求步数
注意到把n - 1个圆盘从一个柱子挪到另一个柱子的步数和从哪个柱子挪到哪个柱子无关，是固定的所以可以用递推

数学求步数

计算组合数

组合数C(m, n) = m! / (n! * (n - m)!)，原始公式涉及到三个阶乘，不但计算复杂，还容易爆long long，利用公式C(m, n) = C(m, n - 1) + C(m - 1, n - 1)就可以利用记忆化+递归，或递推快速不爆上界地求出组合数

  4. Pivot is the first nonzero in the row that does the elimination.
    The multiplier is the entry to eliminate divided by pivot.
  5. When solving the system, right-hand side vector and the coefficients act the same. So it is convenient to add it to the coefficient matrix. The matrix with the right-hand side vector is called the augmented matrix, written as .
  6. After elimination, we back substitute from bottom to top to solve each unknowns.
  7. Elimination can fail when we do not get $n$ pivots. This could lead to either no solution or infinitely many solutions. However, if there is a nonzero below the zero in the pivot, we can exchange them to correct the system.

Matrix Elimination

Operation by Rows or Columns

  1. Every matrix is a series of row vectors or column vectors. When multiplying two matrices, we can multiply them separately. We can multiply the first matrix to every column vector in the second matrix or multiply the second matrix to every row vector in the first matrix
  2. This gives us a hint of row operations and column operations. The first matrix can change the second matrix on rows while the second matrix changes the first matrix on columns.
  3. In the multiplication of $LR = A$, the $i$th row of $A$ is $L$ times the $i$th row of $R$ while the $i$th column of $A$ is the $i$th column of $L$ times $R$.

01. The Geometry of Linear Equations

Posted on 2019-10-20 Edited on 2020-04-01 In Mathematics , Linear Algebra (MIT 18.06)

Linear Combination

  1. Vector addition : $\vec{c}=\vec{a}+\vec{b} (c_i= a_i+b_i)$
  2. Scalar multiplication: $\vec{b}=c\vec{a} (b_i=ca_i)$
  3. Linear combination: the sum of $c\vec{a}$ and $d\vec{b}$, i.e. $c\vec{a}+d\vec{b}$
  4. Expressions of vectors: $\vec{x}=\left[\begin{matrix}a \ b \ …\end{matrix}\right]=(a, b, …)≠[a, b, …]$
  5. For one vector $\vec{u}$, the only linear combination is the scalar multiplication $c\vec{u}$.
  6. If all vectors are $\vec{0}$, every combination would be zero.
  7. If $\vec{0}$ does not exist, all the combinations of $n$ independent vectors form an $n$-dimensional space with $\vec{0}$ in it.
  8. The picture of a vector is an arrow pointing from the origin point to the point with the coordination of the vector. Moreover, addition is merely following all the vectors and pointing from the origin to the final point. Multiplication is the arrow $c$ times longer. If $c<0$, the arrow goes the other way.

Lengths and Dot Products

  1. Dot product (Inner product): $\vec{v}\cdot \vec{w}=\sum{v_iw_i}$
  2. If the dot product equals to 0, two vectors are perpendicular. The $\vec{0}$ is perpendicular to every vector.
  3. Length of a vector: $||\vec{v}||=\displaystyle\sqrt{\vec{v}\cdot \vec{v}}=\sqrt{\sum{v_i^2}}$
  4. The length of a vector is the length of the arrow that represents the vector in the picture.
  5. A unit vector $\vec{u}$ is a vector whose length is $1$.
  6. We can create a unit vector by dividing a vector with its length, i.e. $\vec{u}=\displaystyle\frac{\vec{v}}{||\vec{v}||}$.
  7. The standard unit vectors along the $x$ and $y$ axes are written $\vec{i}$ and $\vec{j}$.
  8. $\vec{v}\cdot \vec{w} > 0 \Longleftrightarrow \theta <\frac{\Pi}{2}$
    $\vec{v}\cdot \vec{w} < 0 \Longleftrightarrow \theta > \frac{\Pi}{2}$
  9. For unit vectors, the dot product of two vectors is . If not both vectors are unit vectors, divide them with their length to make them unit vectors.

  11. Triangle Inequality: $||\vec{v}+\vec{u}||≤||\vec{v}||+||\vec{u}||$. Because $\vec{v}+\vec{u}$ is the third side of the triangle of $\vec{u}$ and $\vec{v}$
  12. For $2$-dimensional vectors $\vec{v}=(a, b)$ and $\vec{u}=(b, a)$, the Schwarz Inequality becomes $2ab ≤ a^2 + b^2$. If we let $x=a^2$ and $y=b^2$, $\displaystyle\sqrt{xy}≤\frac{x+y}{2}$, which is Geometric mean ≤ Arithmetic mean

Matrix

Matrix Times Vector

  1. For $A\vec{x}$, one way is to calculate by columns, and the other is by rows.
  2. For columns, the result is the linear combination of each column.
  3. For rows, the result is the dot product of each row and the vector.

Solve Equations

  1. First, we need to make sure those equations are linear, which means that the unknowns are only multiplied by numbers.
  2. For $A\vec{x}=\vec{b}$, we can solve this equation to $\vec{x}=A^{-1}\vec{b}$. Here $A^{-1}$ is the inverse of $A$. $A$ is called the coefficient matrix.
  3. $A\vec{x}=\vec{b}$ has only one solution as long as $\vec{b}$ is in the column space of $A$. Alternatively, this system has either no solution or infinite solutions.
  4. Independence: If no combination of vectors is $\vec{0}$ except all the scalar is 0, this system of vectors is independent.
    Dependence: If at least one combination of vectors is $\vec{0}$, this system of vectors is dependent.
  5. If $A$ has independent columns, $A\vec{x}=\vec{0}$ has one solution. $A$ is an invertible matrix.
    If $A$ has dependent columns, $A\vec{x}=\vec{0}$ has many solutions. $A$ is a singular matrix.

Two ways to see equations

  1. For $n$ equations in $m$ unknowns, We can write them in equation form or the matrix form.
  2. The matrix form has three matrices, the coefficient matrix A, the unknown matrix $\vec{x}$, and the right-hand side matrix $\vec{b}$. These make the system $A\vec{x} = \vec{b}$.
  3. The equation form is the usual form, like “$2x - y = 0$”, which is the row form of the matrix.
  4. According to two ways to calculate the multiplication of matrix and vector, we have two ways to see a system of equations.

Row Picture

  1. In the row picture, we look at the coefficient matrix by rows.
  2. Each row dot products with the unknown matrix and produces an equation which stands for a space.
  3. Furthermore, the answer to the system is the intersection of all spaces.
  4. The row picture of two equations in two unknowns shows two lines meeting at a single point (the solution).

Column Picture

  1. In the column picture, we look at the combination of the column vectors.
  2. Each column, including the right-hand side vector, stands for a vector.
  3. Moreover, the unknowns are the coefficients of the vectors in the linear combination to produce the right-hand-side vector.
  4. The column picture combines the column vectors on the left side to produce the vector $\vec{b}$ on the right side.
  5. The pictures of the higher dimension are hard to draw, even to imagine. Nevertheless, base on the column picture, we can solve other questions.
  6. The question of solving equations is turned into finding linear combinations.
  7. Furthermore, the existence of answers becomes a vector space problem, which becomes the rank problem.
